FixIt – Smart Issue Reporting Platform
FixIt is a web-based platform that allows users to report and track real-world issues in their locality such as broken streetlights, potholes, garbage problems, and more. The goal is to bridge the gap between citizens and authorities by making issue reporting simple, fast, and transparent.

💡 Key Features
📝 Easy Issue Reporting
Users can quickly submit issues with title, description, category, and location.
📍 Location-Based Tracking
Issues are linked to specific locations for better visibility and management.
📊 Real-Time Issue Display
All reported problems are displayed dynamically on the platform.
🔍 Category Filtering
Users can filter issues like Lighting, Roads, Cleanliness, etc.
⚡ Fast & Responsive UI
Built with modern frontend tools for smooth performance.

🛠️ Tech Stack
Frontend: React + Vite
Deployment: Render

🎯 Purpose
The main aim of FixIt is to:
Improve civic engagement
Make issue reporting accessible to everyone
Provide a centralized system for tracking local problems
